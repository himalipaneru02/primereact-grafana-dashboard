# PrimeReact Grafana Dashboard Replica

## Overview
Replicated Grafana Node Exporter dashboard using React + PrimeReact.

## Tech Stack
- React (Vite)
- PrimeReact
- PrimeFlex
- Chart.js

## Features
- Gauge panels
- Metrics stat boxes
- 2x2 chart grid
- Collapsible sections
- Grafana-style sidebar
